Frequently Asked Questions

How does FlappyPDF work without uploading files?

▼

FlappyPDF uses PDF.js and pdf-lib — JavaScript libraries that run directly in your browser. Your files are processed using your own device's memory and CPU, so nothing is ever transmitted to a server.

Will my images be resized or compressed?

▼

No. Images are embedded at their original dimensions and resolution. Each PDF page is sized to match its image exactly, so there's no cropping, stretching, or quality loss.

Is there a file size or page count limit?

▼

There's no hard limit set by FlappyPDF, but very large files (over 100MB) may be slow depending on your device's memory. For best results, we recommend keeping total file size under 150MB.
